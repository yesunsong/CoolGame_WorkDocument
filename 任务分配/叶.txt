可消耗元宝/绑定元宝的功能，当元宝和绑定元宝总和数量不足时：
	武魂强化   材料不足时，弹出元宝购买框时，点击购买，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	兵魂   修炼与进阶  材料不足时，弹出元宝购买框时，点击购买，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	天翼   修炼与进阶 材料不足时，弹出元宝购买框时，点击购买，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	挑战副本 黄金钥匙购买弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	死亡复活 点击原地复活时，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	传送（任务传送、世界传送）  点击传送，出现是否元宝直接购买，点击确定，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。
	悬赏令 秒杀和一键完成  点击弹出是否消耗元宝完成任务，点击确定时，弹出元宝不足提示框：“元宝不足，是否前往充值”，点击充值，跳转至充值界面。

if (!CommandManager.getInstance().checkIngotCommand(trackCost, this)) {

可消耗元宝/绑定元宝的功能，当元宝和绑定元宝总和数量不足时：
在相应的逻辑位置添加如下代码：
if (!CommandManager.getInstance().checkTotalIngotCommand(trackCost, this)) {
			return;
		}
